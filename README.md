## :floppy_disk: Setup

* [Install Poetry](https://python-poetry.org/docs/#installation) as Virtual Env
  * create a virtual env with Poetry: `poetry init`
  * access the newly created virtual env: `poetry shell`
* Install Rembg: `poetry add rembg`
